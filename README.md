# Workify

Workify is an offline-first appointment and business management application designed for professionals to manage their daily operations efficiently.

## Features
- Dashboard and analytics
- Customer management
- Appointment scheduling
- Visit tracking
- Tasks and reminders
- Earnings tracking
- Folder-based organization
- Offline support
- Progressive Web App (PWA)
- Responsive design

## Tech Stack
- HTML5
- CSS3
- JavaScript
- Service Workers
- Web App Manifest
